# Hyprland Index

## Entrypoints
- `common.nix` — shared enablement/import wiring.
- `hyprland-lua.nix` — generated Lua config, input, keybindings.
- `laptop.nix` — laptop monitor/layout snippet.
- `workstation.nix` — workstation monitor/layout snippet.

## Adjacent
- `hyprpaper.nix`, `hypridle.nix`, `hyprsunset.nix` — Hyprland-adjacent services.
- `wallpapers/` — wallpaper assets.
