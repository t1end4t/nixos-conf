Use this guide when working under `home-manager/gui/`. See `INDEX.md` for the navigation map.

## Scope
- User GUI, theme, terminal, notification, launcher, and compositor-adjacent config.

## Constraints
- Keep compositor-specific config in `hyprland/`.
- Keep bar config in `waybar/`.
- Keep terminal-specific config in the matching terminal file.
- Put only GUI imports in `default.nix`.
