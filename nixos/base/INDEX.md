# NixOS Base Index

## Files
- `default.nix` — shared imports.
- `preset.nix` — shared NixOS defaults.
- `environment.nix` — system packages/environment.
- `fonts.nix`, `catppuccin.nix` — fonts/theme.
- `apps/INDEX.md` — shared app modules map.
- `daemon/INDEX.md` — shared daemon modules map.
- `hosts/INDEX.md` — generated hosts-file setup map.
