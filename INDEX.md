# Index


## Start Here
- `flake.nix` — inputs, overlays, NixOS/Home Manager host wiring.

## Top-Level Folders
- `nixos/` — shared and host-specific NixOS config.
- `home-manager/` — shared and host-specific Home Manager config.
- `modules/` — reusable option modules.
- `overlays/` — nixpkgs overlays.
- `pkgs/` — local package definitions.
- `scripts/` — setup and helper scripts.
